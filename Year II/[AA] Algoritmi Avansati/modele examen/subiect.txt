a) Codificarea cromozomului
Un cromozom este o secvență binară de lungime n (în cazul nostru n = 8), unde fiecare genă corespunde unui obiect.

Lungimea cromozomului: n = 8

Valoarea fiecărei gene: 0 (obiectul nu este selectat) sau 1 (obiectul este inclus în soluție).

Exemplu:
Cromozomul 10101001 înseamnă:

Obiectele 1, 3, 5 și 8 sunt selectate.

Restul nu sunt.

✅ b) Funcția de fitness
Funcția de fitness trebuie să:

Maximizeze suma valorilor obiectelor selectate, dar

Penalizeze soluțiile în care probabilitatea totală e sub pragul P.

Definiții:
Fie V[i] valoarea obiectului i

Fie P[i] probabilitatea ca obiectul i să ajungă intact

Pentru o soluție X = x1 x2 ... xn, calculăm:

Valoarea totală: val(X) = Σ xi * V[i]

Probabilitate totală (produs): prob(X) = ∏ P[i]^xi

Fitness:
text
Copy
Edit
fitness(X) = val(X), dacă prob(X) ≥ P
             0       , altfel
Alternativ, putem penaliza:

text
Copy
Edit
fitness(X) = val(X) * min(1, prob(X)/P)

📌 1. Reamintire: Ce sunt semidreptele într-o diagramă Voronoi?
O semidreaptă într-o diagramă Voronoi apare pentru un punct de pe margine – adică dacă o regiune Voronoi este neînchisă, marginea ei se extinde la infinit.

Numărul de semidrepte = numărul de regiuni Voronoi neînchise = numărul de puncte aflate pe convex hull (conturul convex)
