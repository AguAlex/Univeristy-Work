CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML. It controls the layout, colors, fonts, and other visual aspects of a web page, allowing for better presentation and separation of content and design.

Basic Syntax

CSS rulesets consist of a selector and a declaration block. The selector specifies which elements the styles should apply to, and the declaration block contains one or more declarations, each consisting of a property and a value.

selector {
    property: value;
}

For example:

h1 {
    color: blue;
    font-size: 24px;
}

CSS Selectors

CSS selectors are patterns used to select elements to which a set of CSS rules apply. Some common CSS selectors include:

  • Type selectors: h1, p, div
  • Class selectors: .classname
  • ID selectors: #idname
  • Descendant selectors: parent child
  • Pseudo-classes: :hover, :nth-child()

Box Model

The CSS box model describes the rectangular boxes that are generated for elements in the document tree. It consists of content, padding, border, and margin.

The properties related to the box model include:

  • width and height: Specifies the width and height of the content area
  • padding: Clears an area around the content
  • border: A border surrounding the padding
  • margin: Clears an area outside the border
