JavaScript Basics

What is JavaScript?

JavaScript is a programming language that enables you to create dynamically updating content, control multimedia, animate images, and much more on a web page. It is a core technology of the web, along with HTML and CSS.

Basic Syntax

JavaScript syntax is similar to other programming languages like C++ and Java. Here's a simple example of JavaScript code:

// Define a variable
var message = "Hello, world!";

// Display an alert box with the message
alert(message);

Data Types

JavaScript supports various data types, including:

  • String
  • Number
  • Boolean
  • Array
  • Object
  • Null
  • Undefined

Functions

Functions are reusable blocks of code that perform a specific task. They are defined using the function keyword. Here's an example:

function greet(name) {
    return "Hello, " + name + "!";
}

var greeting = greet("John");
console.log(greeting); // Output: Hello, John!

DOM Manipulation

The Document Object Model (DOM) is a programming interface for web documents. JavaScript can be used to manipulate the DOM, enabling dynamic interaction with the HTML and CSS of a web page.
